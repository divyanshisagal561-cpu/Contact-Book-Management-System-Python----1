# Contact Book Management System - Python -- 1

A simple command-line based **Contact Book Management System** built using Python. This project allows users to store, search, delete, and view contacts using Python dictionaries.

---

## Features

* Add a new contact
* Search for an existing contact
* Delete a contact
* Display all saved contacts
* Exit the application

---

## Technologies Used

* Python 
* Dictionary Data Structure
* Loops and Conditional Statements

## How It Works

The program stores contact names and phone numbers in a Python dictionary.

### Menu Options

```text
1. Add Contact
2. Search Contact
3. Delete Contact
4. Show All Contacts
5. Exit
```

### Add Contact

Allows users to save a contact name and phone number.

### Search Contact

Searches for a contact by name and displays the phone number if found.

### Delete Contact

Removes a contact from the contact book.

### Show All Contacts

Displays all saved contacts and their phone numbers.

### Exit

Closes the application.
